CyberRecon
Web Reconnaissance & Vulnerability Scanner

CyberRecon is a modular Python-based web reconnaissance and vulnerability scanner designed for ethical hackers, bug bounty hunters, and cybersecurity researchers. It performs key reconnaissance tasks like DNS lookup, IP geolocation, subdomain enumeration, port scanning, WHOIS lookups, and technology stack detection, all from a single interface.

Features

DNS Record Enumeration (A, AAAA, MX, NS, TXT)
Subdomain Enumeration (passive)
IP Geolocation & ASN Info
Port Scanning (TCP)
WHOIS Lookup
Tech Stack Detection (Wappalyzer)
XSS Injection Testing (via Burp Suite)
Burp Suite Proxy Integration (manual testing)
Modular design for future vulnerability checks

Roadmap

Basic SQLi test integration
Local dashboard with Streamlit/Flask
PDF report generation
Authentication-aware scans
